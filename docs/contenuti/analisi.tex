\section{Analisi}

\subsection{Caratteristiche dell'utenza}
Gli utenti che possono accedere al sito si dividono in 3 categorie:
\begin{itemize}
    \item \textbf{Utenti generici}: utenti non registrati o non loggati i quali navigano nel sito osservando le opere fotografiche;
    \item \textbf{Utenti autenticati}: utenti che hanno superato un processo di autenticazione, possibilmente perch\'e interessati alla possibilit\`a di entrare in contatto con il fotografo tramite un apposito form dove possono prenotare un servizio.
    \item \textbf{Amministratore}: il fotografo stesso o un gestore per esso. Dopo l'autenticazione ha accesso ad un cruscotto personale dove pu\`o gestire la Homepage eliminando o modificando le foto esistenti oppure caricandone di nuove. Inoltre pu\`o gestire anche le prenotazioni effettuate dagli utenti.
\end{itemize}

\subsection{Funzionalit\`a}
Tutti gli utenti del sito possono:
\begin{itemize}
    \item Visualizzare le fotografie nella Homepage, divise per categorie;
    \item Interagire con le opere in Homepage leggendone il titolo, la descrizione oltre alla data e luogo di scatto;
    \item Leggere la storia del fotografo nella pagina ``About";
    \item Stampare le pagine se lo ritengono opportuno.
\end{itemize}
Per gli utenti che hanno effettuato l'accesso semplice \`e inoltre possibile:
\begin{itemize}
    \item Prenotare un servizio fotografico;
    \item Visualizzare le prenotazioni effettuate;
    \item Modificare le prenotazioni effettuate;
    \item Cancellare le prenotazioni effettuate;
    \item Eseguire il logout.
\end{itemize}
Infine l'amministratore del sito pu\`o:
\begin{itemize}
    \item Gestire le prenotazioni ricevute;
    \item Caricare nuove immagini nella Homepage;
    \item Eliminare immagini esistenti nella Homepage;
    \item Modificare la visibilità di immagini esistenti nella Homepage rendendole visibili o meno;
    \item Modificare le informazioni delle immmagini esistenti nella Homepage;
    \item Eseguire il logout.
\end{itemize}

\subsection{Target audience}
Il sito si propone di raggiungere potenziali clienti in cerca di  un fotografo per shooting, eventi (ad esempio concerti, matrimoni, ecc.) o qualunque attività fotografica professionale.
Esempi di possibili ricerche che possono condurre al sito sono:
\begin{itemize}
    \item fotografo eventi professionista
    \item fotografo per matrimoni
    \item fotografo ritratti e shooting
\end{itemize}
